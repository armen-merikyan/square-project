Artist Statement

This piece encapsulates the emotional tension of vulnerability entangled with strength. The palette features earthy tones like moss green and rust, symbolizing resilience and fragility. The color placements create a stark contrast, with the center embodying strength through dense, saturated colors that collide with the softer, more exposed edges representing vulnerability. The arrangement suggests a struggle between the inner core and the surrounding space, where tension builds in identifiable 'pulse marks' against a static background, creating visual depth and movement.
